¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Utiliza etiquetas para estructurar el contenido: títulos, párrafos, imágenes, enlaces, tablas y más.

¿Qué es CSS?

CSS (Cascading Style Sheets) permite dar estilo a las páginas HTML. Con CSS puedes cambiar colores, tamaños, fuentes, márgenes, e incluso animaciones.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. Se usa para validar formularios, crear menús dinámicos, mostrar alertas y más.
