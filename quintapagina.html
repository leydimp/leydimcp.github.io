¿Qué es una tabla en HTML?

Una tabla en HTML se utiliza para mostrar datos organizados en filas y columnas. Es útil para representar información como horarios, precios o listas detalladas.

Ejemplo de tabla de estudiantes

Nombre Carrera Edad Nota Final
Juan Perez Ingeniería de Software 20 18
Ana lopez Seguridad 22 17
Carlos Ramos Mecatrónica 21 19

¿Sabías que?

HTML también permite combinar celdas, usar colores y aplicar estilos con CSS para mejorar la apariencia de tus tablas.

Consejo del día

¡Siempre estructura tu HTML correctamente usando etiquetas semánticas como <article>, <section> y <footer> para una mejor accesibilidad!
